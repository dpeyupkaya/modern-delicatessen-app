🚀 Kurulum

1. Depoyu klonlayın:
```bash
git clone https://github.com/kullanici-adi/DelicatessenHub.git
```

2. Bağımlılıkları yükleyin:
```bash
npm install
```

3. Geliştirme sunucusunu başlatın:
```bash
npm run
```
